# NZGrapher
Welcome to the official source code repository for **NZGrapher**.

NZGrapher is a web based graphing tool. NZ Grapher was designed for New Zealand Schools by a New Zealand Teacher

This is the files that are used to publish NZGrapher
You can check it out live at [https://grapher.nz/](https://grapher.nz/). You can also see more info at [https://info.grapher.nz/](https://info.grapher.nz/)

NZGrapher has been developed by Jake Wills, a maths teacher in New Zealand specifically for supporting the teaching of the NCEA Statistics Standards. The idea behind NZGrapher was to be able to run on any device, without an install. NZGrapher was developed to run on anything with a browser, computers, iPads, ChromeBooks, Microsoft Surface, Android, even Phones.

## 🎓 Why is this code public?
NZGrapher is a commercial product, but I believe in the power of curiosity. This repository exists so that **New Zealand students and educators** can:
* **Peek under the hood:** See how a real-world data visualization tool is constructed.
* **Learn by example:** Understand how we handle datasets, statistical calculations, and graphing logic.
* **Experiment:** Run a local copy to test out new features or ideas for school projects (like NCEA Computer Science internals).
