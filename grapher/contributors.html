Contributors


NZGrapher has been developed by Jake Wills, a maths teacher in New Zealand specifically for supporting the teaching of statistics in New Zealand.
Several others have also contributed code to NZGrapher

You can see the source code for NZGrapher on GitHub.
This is to allow students to explore and learn, but please note that NZGrapher is source-avaiable, not open source. More details on the GitHub repo.

If you have any questions or suggestions, please contact me.
